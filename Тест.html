Домашнее задание — Блог

Задача: сделать блог из предоставленной вёрстки.

Результат: на тестовую площадку загружен блог. Посты берутся из базы данных.

Минимальный функционал:

  • На главной странице отображаются посты.
  • Заголовок и текст поста берутся из базы данных.
  • Детальная страница поста с текстом из базы данных. В базе можно хранить вёрстку с абзацами и картинками.

Дополнительный функционал:

  • Посты отображаются в порядке от новых к старым. Для даты и времени в MySQL существует тип DATETIME.
  • Дата поста форматируется так же, как в вёрстке. Для названия месяца можно создать функцию.
  • Авторы берутся из базы данных.
  • Есть страница с постами автора (author.html в вёрстке).
  • Теги берутся из базы данных.
  • Есть страница с постами по тегу (tag.html в вёрстке).
  • Работает форма на странице контактов. Отправленные формы нужно сохранять в базу данных.
  • "Опубликовал" или "Опубликовала" в зависимости от пола автора.
  • Детальные страницы постов доступны по красивым адресам. Например "/posts/ratatui/".

Также будут оцениваться:

  • Безопасность. Никогда нельзя доверять пользовательскому вводу.
  • Структура проекта. Разделите код по файлам и папкам.
  • Качество кода. Сюда же относятся лишние запросы в базу данных.
  • Дополнительный плюс будет за использование MVC и роутинга.

Откуда взять наполнение для постов? Вы можете продолжить кулинарную тему, заполнить остальные посты Lorem Ipsum или написать что-то своё.
